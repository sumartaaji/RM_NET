# **App Name**: RM_NET Cashier

## Core Features:

- POS Interface: Simple and intuitive Point of Sale (POS) interface for fast transaction processing.
- Product Catalog: Product catalog with search and categorization for easy selection of ATK items.
- Cart Management: Cart management to easily add, remove, and modify the order.
- Payment Processing: Multiple payment method handling and support such as cash and card.
- Sales History: Record keeping functionality for auditability purposes
- Suggested Items: AI-powered tool for suggesting related or popular items to the cashier based on the items already in the cart to help boost sales.

## Style Guidelines:

- Primary color: Soft blue (#64B5F6) to convey a sense of trustworthiness and calm, important for financial apps.
- Background color: Light gray (#F0F4F8), a subtle background to ensure that data presented to the cashier can be clearly and quickly viewed and understood.
- Accent color: Yellow (#FFCA28) for action items like 'charge' button. This analogous color provides visual interest, without distracting from the primary content.
- Body and headline font: 'Inter' (sans-serif) for a clean, modern, readable interface.
- Use consistent, minimalist icons from a standard library (e.g., Material Design Icons) to represent common actions and categories.
- A responsive layout is to be used to guarantee that it adapts well to desktop and tablet screens.
- Subtle animations and transitions should be implemented throughout to provide feedback, confirm actions, and provide additional visual interest without overwhelming users.
